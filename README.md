# Personal Website

Welcome to my personal website, built using Next.js, Tailwind CSS, and TypeScript. It serves as both my portfolio and blog, focusing on topics related to programming and web development.

## Features

-   **Blog**: Explore articles on various aspects of programming and web development, written in Markdown with syntax highlighting for code snippets.
-   **Project Showcase**: View my latest projects, each detailed with descriptions, tags, and GitHub repository links.
-   **About Me**: Get to know me better through a brief introduction and background information.
-   **SEO Optimization**: The website is optimized for search engines with meta tags and semantic HTML.
-   **Responsive Design**: Enjoy a seamless experience across devices with responsive design principles.

## Technologies Used

-   [Next.js](https://nextjs.org/) - A React framework for building server-rendered and static websites.
-   [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for rapidly building custom user interfaces.
-   [TypeScript](https://www.typescriptlang.org/) - A typed superset of JavaScript that compiles to plain JavaScript.
-   [Markdown](https://www.markdownguide.org/) - A lightweight markup language for formatting plain text.
-   [Shiki](https://shiki.matsu.io/) - A code syntax highlighting library used for rendering code snippets in blog posts.

## Getting Started

To run this project locally:

1. Clone the repository:

    ```bash
    git clone https://github.com/pyyupsk/personal-website.git
    ```

2. Install dependencies:

    ```bash
    cd personal-website
    npm install
    ```

3. Start the development server:

    ```bash
    npm run dev
    ```

4. Open [`http://localhost:3000`](http://localhost:3000) in your browser to see the website.
