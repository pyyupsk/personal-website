# Personal Website

This project is my personal website built with Next.js, Tailwind CSS, and TypeScript. It's a simple blog that showcases my projects and provides some information about myself.

## Features

-   **Blog**: This project includes a simple blog that I regularly update with new posts about programming and web development.
-   **Project Showcase**: The website includes a showcase of all of my projects, with a link to each one's GitHub repository.
-   **SEO Optimization**: The website is optimized for search engine optimization, with meta tags and semantic HTML.

## Installation

To run this project locally, you'll need to have Node.js and npm installed on your machine.

1.  Clone this repository.
2.  Run `npm install` to install the project dependencies.
3.  Run `npm run dev` to start the development server.
